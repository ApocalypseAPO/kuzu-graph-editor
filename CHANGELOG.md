# Changelog

本文件记录当前仓库作为二次开发项目的变更历史。

> 说明：
> - 本项目基于 JSONEditor 进行二次开发
> - 上游项目的完整历史记录请参考其官方仓库
> - 此处仅记录当前仓库自身的功能调整与衍生改动

---

## [0.1.0] - 2026-03-10

### Added
- 基于 JSONEditor 搭建节点属性编辑界面
- 新增面向图数据库场景的独立搜索栏
- 新增按节点 `name` 字段进行模糊搜索的功能
- 新增基于 Levenshtein 距离的候选结果排序
- 新增搜索结果点击后自动定位节点
- 新增节点高亮闪烁反馈
- 新增图谱节点与关系的基础编辑辅助功能
- 新增 Python 后端接口用于搜索请求处理
- 新增图数据库查询封装逻辑

### Changed
- 重组前端页面布局，以适配图谱总览 + JSON 编辑的双栏使用方式
- 关闭 JSONEditor 默认顶部菜单、导航栏和状态栏
- 对关键字段 `id`、`table`、`entityType` 增加只读限制

### Notes
- 当前版本主要用于课程作业展示与非商业场景下的工作辅助
- 本项目是 JSONEditor 的非官方衍生项目
